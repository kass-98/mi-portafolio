Para descargar el proyecto y verlo en tu computadora, estos son los pasos que debes seguir:

1. Entra al repositorio en GitHub.
2. Da clic en el botón verde que dice Code.
3. Selecciona Download ZIP.
4. Cuando se descargue, descomprime la carpeta.
5. Abre Visual Studio Code.
6. Da clic en File > Open Folder.
7. elecciona la carpeta que descomprimiste.
8. Abre el archivo index.html y listo
